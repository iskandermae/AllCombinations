# AllCombinations<br>
Return all combination of elements  (their indexes) in a given array<br>
<br>
<br>
Example for an array of 4 elements:<br>
0 0 0 1<br>
0 0 0 2<br>
0 0 0 3<br>
0 0 0 4<br>
0 0 1 2<br>
0 0 1 3<br>
0 0 1 4<br>
0 0 2 3<br>
0 0 2 4<br>
0 0 3 4<br>
0 1 2 3<br>
0 1 2 4<br>
0 1 3 4<br>
0 2 3 4<br>
1 2 3 4<br>
